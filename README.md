# 디스코드 챗봇
디스코드에서 챗봇을 이용하여 유튜브 음악을 검색하고 재생할 수 있습니다.
다음과 같은 기능이 있습니다.
* 음악검색
* 음악재생/일시정지 및 재생목록에 추가/제거
* 날씨검색 등

## 사용방법
디스코드에서 봇 생성후, 만들어진 토큰을 코드 내부에 입력하여 실행합니다.
봇이 채팅채널에 초대되면, !<명령어> <옵션> 을 입력하여 사용합니다.
ex) !음악검색 애국가
    !챗봇
